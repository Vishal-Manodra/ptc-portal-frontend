# PTC Portal - Frontend

This is the Frontend client for the PTC Portal application. It provides an intuitive interface for both the firm employees (Admins) and clients to manage operations, tasks, and file compliance requirements.

## 🚀 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router
- **State Management / Data Fetching**: React Query (@tanstack/react-query)

## 📦 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

The app will be accessible at `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

This will generate a `dist/` folder with static files ready for deployment.
